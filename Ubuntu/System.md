System
===============================

Process
-------------------------------
View memory info
```
cat /proc/meminfo
```

LSB
--------------------------------
- Linux Standards Base (LSB)

If not already installed
```
sudo apt install lsb-release
```

Get all OS version information
```
lsb_release -a
```


Hardware
---------------------------------

Reboot
```
sudo reboot
```

Shutdown
```
sudo shutdown
```

Restart
```
sudo shutdown -r
```
